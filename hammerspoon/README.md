# hammerspoon/

## Purpose
Hammerspoon automation config and bundled Spoons.

## Contents
- `init.lua` — Entry-point Hammerspoon config.
- `Spoons/` — Local spoon modules:
  - `AClock.spoon/`
  - `Calendar.spoon/`
  - `GoMaCal.spoon/`
  - `HCalendar.spoon/`
- `README.md` — Directory documentation.

## Current Stow target result
- `~/.config/init.lua`
- `~/.config/Spoons/...`

## Usage status
- Potentially mis-targeted. Hammerspoon typically loads from `~/.hammerspoon/init.lua`.
- May work only if you also point Hammerspoon to this location.
