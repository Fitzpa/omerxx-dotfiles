# ghostty/

## Purpose
Configuration and visual assets for Ghostty terminal.

## Contents
- `config` — Main Ghostty configuration file.
- `ps.json` — Additional Ghostty-related JSON settings.
- `shaders/space.glsl` — Shader asset.

## Current Stow target result
- `~/.config/config`
- `~/.config/ps.json`
- `~/.config/shaders/space.glsl`

## Usage status
- Likely partially unused unless Ghostty is configured to load from these exact paths.
- Typical layout is `~/.config/ghostty/config` with assets under that same namespace.
